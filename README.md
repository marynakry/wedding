# Wedding Invitation — One‑page React + TypeScript site

Quick scaffolded app using Vite, React, and TypeScript.

Setup

```bash
cd wedding
npm install
npm run dev
```

Build

```bash
npm run build
npm run preview
```

Language

- The site supports both RU and EN.
- Use the toggle at the top of the card to switch language.

GitHub Pages

- This project is configured for project pages at `https://marynakry.github.io/wedding`.
- Vite base is set to `/wedding/` in `vite.config.ts`.
- Deployment runs automatically via GitHub Actions workflow `.github/workflows/deploy.yml` on push to `main`.
